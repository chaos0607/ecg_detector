Detailed Description

This class reads ECG data from a file, processes it in real-time to calculate heart rate, and stores the results in a thread-safe queue.

void OnlineData::generate_data ( )

This method runs an infinite loop that reads, processes, and enqueues ECG data points. Each processed data point is added to the queue, and a notification is sent to waiting threads.
